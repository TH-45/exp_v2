# 基于B/S架构的招投标全流程协同管理系统功能模块

## 招投标管理服务

| 子模块 | 核心功能描述 |
|--------|-------------|
| **投标管理** | • 投标信息录入、编辑、查询、导出<br>• 资质信息一键关联投标<br>• 投标进度跟踪 |
| **招标管理** | • 招标公告发布<br>• 招标文件上传<br>• 投标人资格审核<br>• 评标流程记录 |
| **合同管理** | • 合同起草、审核、签署、归档<br>• 合同与招投标项目自动关联<br>• 合同履约跟踪 |

## 组织与权限服务

| 子模块 | 核心功能描述 |
|--------|-------------|
| **部门与组织架构管理** | • 部门新增、删除、编辑<br>• 组织层级结构维护（集团/公司/部门/项目部等）<br>• 部门负责人设置<br>• 部门启用/停用<br>• 部门与数据权限范围基础关联<br>• 支持树形组织结构展示 |
| **岗位管理** | • 岗位新增、删除、编辑<br>• 岗位职责说明维护<br>• 岗位与部门关联（同一岗位可跨部门使用）<br>• 用户/人员-岗位-部门的关联配置<br>• 岗位默认权限模板（关联推荐角色）<br>• 岗位启用/停用管理 |
| **角色与权限管理** | • 角色新增、删除、编辑<br>• 菜单权限配置（可访问功能模块）<br>• 功能权限配置（新增/编辑/删除/审批等操作）<br>• 数据权限范围配置（本部门/本公司/指定项目等）<br>• 角色授权给账号/人员/岗位<br>• 权限变更记录与审计 |
| **人员信息管理** | • 人员基础信息维护（姓名、工号、联系方式、职务等）<br>• 人员与账号绑定管理（1人1账号为主）<br>• 人员与部门、岗位、角色关联维护<br>• 人员任职与项目参与记录（项目角色、起止时间）<br>• 人员状态管理（在职/离职/停用，联动账号状态）<br>• 人员信息查询与导出 |

## 企业信息服务

| 子模块 | 核心功能描述 |
|--------|-------------|
| **内部信息 - 账号信息** | • 用户账号注册、登录、密码重置<br>• 账号与部门/岗位绑定<br>• 账号状态管理 |
| **内部信息 - 制度与公告信息** | • 制度和公告的发布、修改 |
| **外部信息 - 资质信息** | • 企业资质上传、分类存储、过期提醒<br>• 资质与投标项目关联调用 |
| **外部信息 - 基础信息** | • 企业基本信息维护<br>• 联系方式管理<br>• 合作单位信息录入 |

## 工程项目服务

| 子模块 | 核心功能描述 |
|--------|-------------|
| **人员配置管理** | • 项目团队组建<br>• 人员与项目关联<br>• 岗位职责分配 |
| **物料配置管理** | • 项目物料需求录入、采购关联<br>• 物料使用跟踪、库存预警 |
| **项目进度管理** | • 进度计划制定、里程碑节点设置<br>• 进度更新、延期提醒<br>• 进度可视化展示 |
